## 新得体会

在handler中，体会最深的是message中next这个属性，它将信息巧妙地连接在了一起，除头信息外，消息列表中地所有信息都是它上一个信息的属性，这使得信息排列具有顺序，给人一种第一次用wasd按键进行上下左右移动的美。
